# bootcamp
Code for web dev bootcamp

Libaries/framework used:
Bootstrap - https://getbootstrap.com/docs/5.3/getting-started
Parcel - https://en.parceljs.org/getting_started.html
npm package manager - https://docs.npmjs.com/

To run code locally:

- clone the repo
- execute **npm install**
- execute **npm run dev**
